块级作用域block

通过var声明的变量或者非严格模式下(non-strict mode)创建的函数声明没有块级作用域。2种情况会有块级作用域：{}、label:{}严格模式下，使用let、const声明的变量是有块级作用域的。

经典面试题：6个6

setTimeout() 接受两个参数：

  1. 第一个是包含js代码的字符串和函数，第二个参数是等待多长时间的毫秒数
  2. 第二个参数是指第一段代码经过多少ms后会被加入到队列中。

**返回值：**返回整型数值id,是setTimeout的唯一标识符,可用于取消setTimeout设置的函数clearTimeout(id)。

同步和异步： 同步：可以直接拿到结果（预期返回值）或看到了预期效果 异步：不能直接拿到结果，需要通过一些方法（例如：轮询、回调等）

setTimeout 到底是同步还是异步的 ？→同步！

除了放置异步任务的事件，“任务队列"还可以放置定时事件。

setTimeout()接受两个参数，第一个是回调函数，第二个是推迟执行的毫秒数。需要注意的是，setTimeout()只是将事件插入了"任务队列”，必须等到当前代码（执行栈）执行完，主线程才会去执行它指定的回调函数。要是当前代码耗时很长，有可能要等很久，所以并没有办法保证，回调函数一定会在setTimeout()指定的时间执行。

for(var i=0;i<6;i++){
	setTimeout(
  	()=>{console.log(i)}
  ,0)
}
//输出结果：6个6
//原因：i=6时不符合i<6的条件退出循环，此时循环结束
//当前循环结束后马上执行setTimeout里的这个回调函数，但此时的i=6，所以会打出6个6

如何输出012345？

使用let

for(let i=0;i<6;i++){
	setTimeout(
  	()=>{console.log(i)}
  ,0)
}
//let的作用：就是把当前i记下，不随着i++而改变。作用域就是{}

写立即执行函数

for(var i=0;i<6;i++){
	!function(i)setTimeout(
  	()=>{console.log(i)}
  ,0)(i)
}
//每次进循环体，都会把当时的i赋值给里面的i。里面的i不会i++

var、let、const的区别

  1. var声明的变量处于全局作用域；let、const声明的变量处于块级作用域
  2. 用let重定义变量会抛出一个语法错误：Identifier ‘a’ has already been declared。变量a已经被声明。
  3. const是用来定义常量的。
  • 不能只声明不赋值
  • 不能重新赋值
  • 不能重复声明变量
const a = {a:'a'};//const 定义引用类型时，表示这个对象的地址是不能改变的
a = {a:'b'};//但是对象内部的属性是可以改变的
a.a = 'b'

class

点击查看阮一峰关于class的说明class内部写2个东西：

  • 一个是**constructor(){}**定义了实例的属性，里面的**this**代表实例对象。属性在实例身上。
  • 一个定义类的方法：类的所有方法都定义在类的prototype属性上面。

几点需要注意的：

  1. ​写类的方法时，如果有this，这个this指向类的实例。如果将这个方法提取出来单独使用，this会指向该方法运行时所在的环境（由于 class 内部是严格模式，所以 this 实际指向的是undefined）从而报错

解决办法：

在构造函数上绑定this 在构造函数中使用箭头函数

2.实例的属性可以简写：直接定义在类的最顶层，不用写this

class的继承

点击查看阮一峰关于class继承的说明继承怎么写：

  • 子类使用父类时，必须调用**super()**，其代表了父类的构造函数。
  • 然后再用**constructor()**改造。

先将父类实例对象的属性和方法，加到this上面（所以必须先调用super方法），然后再用子类的构造函数修改this。

let 和 const

let和const都必须先声明，再使用，声明的变量处于块级作用域。 const不能重复声明，声明时必须赋值，且一旦赋值不能改变。 所以一般写代码时，用let声明变量，const声明常量。

字符串模板

使用 ```标识，内部可以使用${}`存放变量，JS表达式，对象的引用以及函数调用等。 在字符串模板中的空格、换行等格式会被保留

…运算符

...运算符可以叫扩展运算符，用来展开一个可迭代对象中的所有项。

//拷贝数组
var arr1 = [1, 2, 3];
var arr2 = [...arr1];

console.log(arr1 === arr2) // false

arr2.push(4); // 修改arr2，不会影响arr1的值

console.log(arr1); // [1, 2, 3]
console.log(arr2); // [1, 2, 3, 4]

//拷贝对象
var obj1 = {
    a: 1,
    b: 2
};

var obj2 = {...obj1};

console.log(obj1 === obj2); // false

obj2.c = 3; // 修改obj2不会影响obj1
console.log(obj1); // {a: 1, b: 2}
console.log(obj2); // {a: 1, b: 2, c: 3}

//字符串转数组
var str = 'hello';
// ES5 处理方式
var es5Arr = str.split('');
console.log(es5Arr) // ["h", "e", "l", "l", "o"]

// ES6 处理方式
var es6Arr = [...str];
console.log(es6Arr) // ["h", "e", "l", "l", "o"]

//类数组转化为数组
const nodeList = document.getElementsByClassName("test");
const array = [...nodeList];
  
console.log(nodeList); //Result: HTMLCollection [ div.test, div.test ]
console.log(array); //Result: Array [ div.test, div.test ]

...还可以叫剩余操作符，把剩余参数合并起来

//如果形参个数太多
let demoFun = function(argA, ...args) {
    console.log(argA);
    console.log(args)
}

demoFun(1, 2, 3);
// 1
// [2, 3]

解构赋值时也经常用到...运算符。

解构赋值

将属性/值从对象/数组中取出,赋值给其他变量。

语法

var a, b, rest;
[a, b] = [10, 20];
console.log(a); // 10
console.log(b); // 20

[a, b, ...rest] = [10, 20, 30, 40, 50];
console.log(a); // 10
console.log(b); // 20
console.log(rest); // [30, 40, 50]

({ a, b } = { a: 10, b: 20 });
console.log(a); // 10
console.log(b); // 20


({a, b, ...rest} = {a: 10, b: 20, c: 30, d: 40});
console.log(a); // 10
console.log(b); // 20
console.log(rest); // {c: 30, d: 40}

使用场景

1.交换变量

let zero = 2;
let one = 1;
let two = 0;

[zero, one, two] = [two, one, zero];

console.log(zero); // 0
console.log(one); // 1
console.log(two); // 2

2.访问可能为空的值/属性

不需要if判断是否为空，直接数组解构给一个默认值

const colors = [];
const [firstColor = 'white'] = colors;
console.log(firstColor); // 'white'

模块化

使用export导出，import导入

//module.js
export let temp="这是一个临时变量";
export let isTrue=false;
export function run(){
   console.log("hellow");
}
//main.js
import { temp, isTrue, run } from "./module";
console.log(temp,isTrue);
run();

//export default默认导出
//module.js
let temp="临时变量";
let isTrue=false;
export default { temp, isTrue };
//main.js
import module from './module'
console.log(module.temp, module.isTrue);

class和继承

class内部写2个东西： ● 一个是constructor(){}定义了实例的属性，里面的this代表实例对象。属性在实例身上。 ● 一个定义类的方法：类的所有方法都定义在类的prototype属性上面。

class Person{
  constructor(name, age){ 
    this.name = name;
    this.age = age;
  }
  sayHi (){
      console.log(`你好，我叫${this.name}`)
      }
}

继承：

//
class Xiaoming extends Person{
  getAge(){
      console.log(`我今年${this.age}岁`)
}
}
let xiaoming = new Xiaoming('小明',18)
xiaoming.sayHi()//output:你好，我叫小明
xiaoming.getAge()//output:我今年18岁
