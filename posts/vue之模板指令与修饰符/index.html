1.模板

Vue 里，模板一般指template，

1.Vue 完整版，写在 HTML 里

<div id="app">
  {{n}}
<button @click="add">+1</button>  
</div>

new Vue({ el:"#app", data:{n:0}, //data可以写成函数 methods:{add(){}} })

2.Vue 完整版，写在选项里

选项里的div替换了上面id=app的div

<div id="app">
</div>

new Vue({ template:`
<div>
  	{{n}}
		<button @click="add">+1</button>  
		</div>
` data:{n:0}, //data可以写成函数 methods:{add(){this.n+=1}} }).$mount("#app")
//el:"#app",替换成.$mount("css选择器")

3.非完整版，配合组件

4.模板中的语法

  1. 展示内容
  • {{obeject.n}}表达式
  • {{n + 1}} 任何运算
  • {{fn(n)}} 可以调用函数（这个函数一般在methods里找）
  • 如果值为undefined或null就不显示
  • 还有一种写法<div v-text="表达式"></div>

​

  1. 展示 HTML

用v-html指令：<div v-html="x"></div>​

  1. 展示{{}}

用v-pre指令：<div v-pre>{{n}}</div>v-pre 不会对模板进行预编译

2.指令

以v-开头的都是指令。语法：v-指令名:参数=值（值没有特殊字符，引号可省略，但建议不省略）有些指令没有参数和值：如v-pre有些指令没有值，如v-on:click.prevent​

除了展示内容，还有更多的指令：

绑定属性

v-bind指令<img v-bind:src="x">简写为<img :src="x">绑定对象（例如 style 样式，也是对象。在 JS 中都是.调用style的）<div :style="{border:1px solid red ,height:100}">px可以省略</div>​

绑定事件

v-on指令：v-on+事件名，如v-on:click=add（值没有特殊字符，引号可省略）除了事件名还可以直接写代码（if else 要用?:代替）缩写@click="事件名"​

条件判断 if else

<div v-if="x > 0">
  x 大于 0
</div>
<div v-else-if="x === 0">
  x 等于 0
</div>
<div v-else="x < 0">
  x 小于 0
</div>

循环 for

用法：for(value,key) in 对象或数组注意：使用v-for后面必须要写:key，不然会报错。一般来说key必须绑定不会重合的值，推荐使用下标

//数组
<ul>
  <li v-for="(value,index) in users" ,:key="index">
    索引：{{idnex}}，值：{{u.name}}
  </li>
</ul>

//对象
<ul>
  <li v-for="(value,name) in obj" ,:key="name">
    属性名：{{name}}，属性值：{{name}}
  </li>
</ul>

显示，隐藏

v-show指令+表达式（用得较少）​

实际上是绑定属性的一种语法糖​

<div v-show="n%2===0">n是偶数</div>

//实际上等同于下面的写法
<div :style={display:n%2===0?'block':'none'}>n是偶数</div>

但是注意：看得见的元素不一定是block​

总结

  • 使用XML而非HTML
  • 使用{{}}插入表达式
  • 使用v-html，v-on，v-bind 等指令操作 DOM
  • 使用v-if，v-for 等指令实现条件判断和循环

指令是一种声明式编程，DOM 操作是一种命令式编程

3.修饰符

有些指令是没有值的，后面是+修饰符比如：

  • @click.stop="add"表示阻止事件传播/冒泡
  • @click.prevent="add"表示阻止事件默认动作
  • @click.stop.prevent="add"同时表示以上 2 种意思

列举下修饰符：**v-on**支持：.{KeyCode|KeyAlias}（监听键盘输入事件@keyPress）.stop表示阻止事件传播/冒泡.prevent表示阻止事件默认动作以及：.capture``.self``.once``.passive``.native​

**v-bind**支持：.sync以及：.props，.camel​

**v-model**支持：.lazy.number.trim​

.sync

其实是组件通信的语法糖​

举例：点击按钮+1在线代码示例在父组件上添加对子组件的事件监听，可以使用.sync

<Button :btnvalue="value" v-on:update:btnvalue="value = $event"></Button>

//等价于
<Button :btnvalue.sync="value"></Button>
//.sync其实是上面这句定义在父组件上事件监听的 语法糖
