最近做项目 Vue3造轮子 时，用到的。顺便记录下。参考Sass中文文档

1.语法格式SCSS

Sass 有两种语法格式。首先是 SCSS (Sassy CSS) —— 也是本文示例所使用的格式 —— 这种格式仅在 CSS3 语法的基础上进行拓展，所有 CSS3 语法在 SCSS 中都是通用的，同时加入 Sass 的特色功能。此外，SCSS 也支持大多数 CSS hacks 写法以及浏览器前缀写法 (vendor-specific syntax)，以及早期的 IE 滤镜写法。这种格式以.scss 作为拓展名。还有一种最早的，是以.sass为后缀名两种格式可以通过sass-convert切换

# Convert Sass to SCSS
$ sass-convert style.sass style.scss

# Convert SCSS to Sass
$ sass-convert style.scss style.sass

2.允许选择器嵌套

优点：避免重复写父级选择器用法：父选择器的样式内部再嵌套子选择器代码示例：

.parent{
	color:#000;
  
  .son{
  	width:20px;
    height:20px;
    background-color: #ff0000;
  }
}

3.&符号代表嵌套外层的父选择器

优点：结构清晰，避免重复写父选择器用法：

  • &代表嵌套规则外层的选择器

代码示例：

  1. 给元素设定**:hover**、**::before**、**::after**这种：
.parent{
	color:#000;
  &:hover{color:red}
}
  1. body有classname时，用&代表嵌套规则外层的选择器
.parent{
	color:#000;
  body.demo &{color:red}
}
  1. 如果含有多层嵌套，最外层的父选择器会一层一层向下传递：
.parent{
	color:#000;
  
  .son{
  	width:20px;
    height:20px;
    background-color: #ff0000;
    
    &:hover{ background-color: blue;}
  }
}

这里的&表示：

.parent .son:hover{background-color: blue;}
  1. &作为选择器的第一个字符，其后可以跟随后缀生成复合的选择器

代码示例：

#main {
  color: black;
  &-sidebar { border: 1px solid; }
}
  • 这里的&-sidebar表示#main-sidebar
  • 当父选择器含有不合适的后缀时，Sass 将会报错。

4.允许属性嵌套

优点：避免重复输入同一类的属性用法：

  • 所有同一类的属性的前缀只用写一次，然后都写到{}里
  • 前缀后面也可以写属性

代码示例：

.funky {
  font: 20px/24px {
    family: fantasy;
    weight: bold;
  }
}

5.注释多了//单行注释

  • 有css原有的多行注释/* */会被编译到css文件里
  • 多了css原本没有的//单行注释，不会被编译到css文件里

6.Sasscript

1.$变量

  • 支持块级作用域
  • 嵌套内部定义的$变量只能在嵌套内部使用
  • 嵌套外部定义的$变量可以在全局使用
  • 将局部变量转换为全局变量可以添加 !global声明

优点：

  • 如果变量赋值是一个像素，那么之后改像素值只用改这个变量，非常省事。

碰壁？这里的span选择器和&:hover > span选择器为啥用不了$h变量？😥做Vue3造轮子中的switch组件时。先用像素写着吧。

<style lang="scss">
$h: 22px;
$h2: $h -4px;
button {
  border: none;
  background-color: blue;
  width: $h * 2;
  height: $h;
  border-radius: $h/2;
  position: relative;
  span {
    position: absolute;
    top: 2px;
    left: 2px;
    width: 18px;
    height: 18px;
    background-color: #fff;
    border-radius: 9px;
  }
  &:hover > span {
    left: calc(100% - #{$h2} - 2px);
  }
}
</style>
